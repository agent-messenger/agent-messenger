## Installation

```bash
npm install agent-messenger
```

> **Note**: The Teams SDK is available for internal use but not yet formally exported as a public entry point in `package.json`. The API may change.

## Available Exports

```typescript
// Internal import path
import { TeamsClient } from 'agent-messenger/src/platforms/teams/client'
import { TeamsCredentialManager } from 'agent-messenger/src/platforms/teams/credential-manager'
import { TeamsTokenExtractor } from 'agent-messenger/src/platforms/teams/token-extractor'
```

### TeamsClient

The main client for interacting with Microsoft Teams' API using user tokens.

### TeamsTokenExtractor

Extracts Teams user tokens from the desktop application automatically. Supports both New Teams and Classic Teams on macOS and Windows.
