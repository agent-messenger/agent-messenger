## Installation

```bash
npm install agent-messenger
```

> **Note**: The Channel Talk SDK is available for internal use but not yet formally exported as a public entry point in `package.json`. The API may change.

## Available Exports

```typescript
// Internal import path
import {
  ChannelClient,
  ChannelCredentialManager,
  ChannelTokenExtractor,
  ChannelError,
} from 'agent-messenger/src/platforms/channeltalk'
```

### ChannelClient

The main client for interacting with Channel Talk's API using extracted user cookies.

### ChannelTokenExtractor

Extracts authentication cookies from the Channel Talk desktop app's SQLite database. Supports macOS (App Store and Electron) and Windows.
