## Installation

```bash
npm install agent-messenger
```

> **Note**: The Channel Talk Bot SDK is available for internal use but not yet formally exported as a public entry point in `package.json`. The API may change.

## Available Exports

```typescript
// Internal import path
import {
  ChannelBotClient,
  ChannelBotCredentialManager,
  ChannelBotError,
} from 'agent-messenger/src/platforms/channeltalkbot'
```

### ChannelBotClient

The main client for interacting with Channel Talk's API using Access Key + Access Secret authentication.
