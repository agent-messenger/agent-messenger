## Installation

```bash
npm install agent-messenger
```

> **Note**: The Telegram SDK is available for internal use but not yet formally exported as a public entry point in `package.json`. The API may change.

## Available Exports

```typescript
// Internal import path
import { TelegramClient } from 'agent-messenger/src/platforms/telegram/client'
import { TelegramCredentialManager } from 'agent-messenger/src/platforms/telegram/credential-manager'
```

### TelegramClient

The main client for interacting with Telegram via TDLib. Requires TDLib (`libtdjson`) to be available — either bundled via `prebuilt-tdlib` or installed manually.

### TDLib Integration

```typescript
// Internal import path
import { TDLib } from 'agent-messenger/src/platforms/telegram/tdlib'
```

The TDLib wrapper handles:
- Loading `libtdjson` shared library
- Managing TDLib client lifecycle
- Sending and receiving TDLib updates
