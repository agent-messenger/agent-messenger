## Installation

```bash
npm install agent-messenger
```

> **Note**: The Discord SDK is available for internal use but not yet formally exported as a public entry point in `package.json`. The API may change.

## Available Exports

```typescript
// Internal import path
import { DiscordClient } from 'agent-messenger/src/platforms/discord/client'
import { DiscordCredentialManager } from 'agent-messenger/src/platforms/discord/credential-manager'
import { DiscordTokenExtractor } from 'agent-messenger/src/platforms/discord/token-extractor'
```

### DiscordClient

The main client for interacting with Discord's API using user tokens.

### DiscordTokenExtractor

Extracts Discord user tokens from the desktop application automatically.
