## Installation

```bash
npm install agent-messenger
```

> **Note**: The WhatsApp Bot SDK is available for internal use but not yet formally exported as a public entry point in `package.json`. The API may change.

## Available Exports

```typescript
// Internal import path
import {
  WhatsAppBotClient,
  WhatsAppBotCredentialManager,
  WhatsAppBotError,
} from 'agent-messenger/src/platforms/whatsappbot'
```

### WhatsAppBotClient

The main client for interacting with the WhatsApp Business Cloud API. Uses Phone Number ID + Access Token authentication.
