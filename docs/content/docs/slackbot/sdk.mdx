## Installation

```bash
npm install agent-messenger
```

> **Note**: The Slack Bot SDK is available for internal use but not yet formally exported as a public entry point in `package.json`. The API may change.

## Available Exports

```typescript
// Internal import path
import {
  SlackBotClient,
  SlackBotCredentialManager,
  SlackBotError,
} from 'agent-messenger/src/platforms/slackbot'
```

### SlackBotClient

The main client for interacting with Slack via bot tokens (xoxb-).
